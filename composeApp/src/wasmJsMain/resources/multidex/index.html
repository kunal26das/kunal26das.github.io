Multidex

A beautiful, offline-friendly Pokédex — on Android, iOS and Desktop.

Multidex is a playful pocket guide to the world of Pokémon. Browse every species, move, type and ability, with artwork-driven theming, multi-language names and flavor text, and smart caching so entries you've opened keep working offline.

Get it on Google Play View source on GitHub

Features

  • Complete Pokédex — species, individual Pokémon, moves, types and abilities from PokéAPI.
  • Offline-friendly — previously opened entries are cached on-device.
  • Multi-language — Pokémon names and flavor text in your language, where available.
  • Shiny artwork — unlock shiny forms with the in-app Shiny Charm.
  • Adaptive design — Material 3 Expressive theming with accents drawn from each Pokémon's artwork, scaling from phone to tablet to desktop.
  • AI descriptions — on-device AI rewriting of descriptions on supported Android devices.

One codebase, three platforms

Multidex is built with Kotlin Multiplatform and Compose Multiplatform — a single Kotlin codebase shipping to Android, iOS and Desktop (JVM). The project is open source under the MIT license; developers are welcome to explore the code on GitHub.
